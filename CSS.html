CSS



O CSS (Cascading Style Sheets) é uma linguagem de estilização que é utilizada para, como o nome diz, estilizar elementos HTML.
De forma muito resumida, a maneira que tu escreves CSS é por regras. Regras que depois vais aplicar em elementos HTML.

Existem diversas formas de escrever CSS. Mas antes de falar sobre isso eu vou mostrar os componentes mais importantes que formam um documento CSS.

Seguinte

Elemento <style> e propriedade "style"


O elemento <style> é um elemento HTML que se usa para poder escrever código Internal CSS
Também existe a propriedade "style". Que coloca-se dentro de um elemento HTML para se escrever Inline CSS



Exemplo de elemento <style>

<html>
<head>

<style>

</style>

</head>
...


Exemplo da propriedade "style"

<html>
<body>

<p style="regra: valor;">exemplo de conteúdo</p>

</body>
...

Seletores


Seletores são usados para defenir um conjunto de regras CSS.
Quando escreves Internal CSS ou External CSS precisas de criar um seletor antes de escrever qualquer regra.


Existem vários tipos de seletores:

  • Seletor TAG
  • Seletor CLASSE
  • Seletor ID

Regras


Regras CSS são propriedades que mudam a forma de um elemento HTML.
O que estas regras fazem é basicamente dizer ao browser como ele deve apresentar o elemento HTML que está a ser afetado por essas regras.

Uma das formas que se escreve uma regra CSS é assim:

<style>
.palavra{

color: #f98b88;

}
</style>
Anterior Seguinte

Valores


Os valores são aplicados a seguir ao nome da regra. E são essenciais para regra ter qualquer utilidade.
Existem regras que precisam de mais do que um valor para funcionar de forma correta.

Pegando o exemplo anterior um valor é representado da seguinte forma:

<style>
.palavra{

color: #f98b88;
border 15px dotted #f98b88;

}
</style>
