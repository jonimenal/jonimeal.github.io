HTML



HTML (HyperText Markup Language) é uma linguagem de formatação que usa <elementos> para a criação de páginas web.
O HTML pode ser escrito até num bloco de notas, o browser lê o código e tranforma-o no que pode ser visto agora mesmo.

Um ficheiro HTML tem 3 elementos principais, que são:

  1. <html>: Representa a raiz de um documento HTML;
  2. <head>: Onde se escreve a parte do HTML que não é vista (como código CSS ou JavaScript...);
  3. <body>: Corpo do documento HTML, onde é colocada maior parte dos elementos e a parte que pode ser vista (tudo o que estás a ver agora faz parte do body)

Propriedades


Dentro dos elementos é possivel adicionar propriedades que mudam a forma como estes elementos são lidos.
Dependendo do elemento HTML, propriedades podem ou não ser essenciais, por exemplo:
O elemento <a> é usado para criar hiperligações que levam o usuário de um sítio a outro. Sem a propriedade "href", o elemento <a> tem basicamente a mesma função de um elemento <p>
Para colocar propriedades é simples! É só escrever a propriedade que quiseres dentro da primeira parte elemento:



<a href="exemplo de um link atoa"> carrega aqui </a>

Conteúdo


Outra parte importante de um elemento HTML é o conteúdo que dentro dele é colocado.
Existem diversos elementos que têm de ter algum tipo de conteúdo escrito dentro para que possam ser visíveis no <body>

Olhando mais uma vez para o exemplo do <a> o conteúdo que está lá escrito é:



<a href="exemplo de um link atoa">carrega aqui</a>

Síntese


A parte príncipal de um documento HTML são os seus elementos
Um elemento HTML pode ser composto por:
O próprio elemento, uma ou mais propriedades, valores para as respetivas propriedades e um conteúdo
