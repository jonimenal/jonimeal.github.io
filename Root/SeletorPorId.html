Seletor ID


O seletor ID escolhe todos os elementos que estejam identificados com o seu nome. A forma como eu escrevo um seletor tipo classe em CSS é colocando um "#" antes do que eu escrevo. Por exemplo: #banana

<style>
#banana{


}
</style>

O seletor ID faz basicamente o mesmo que o seletor CLASSE. Diferente da propriedade class que suporta vários
Para identificar um elemento, eu faço o mesmo que fiz para classificar um elemento. A diferença é que ao inves de usar a propriedade class eu uso a propriedade id

<html>
<head>
</head>
<body>

<p id="banana">

</body>
</html>

A única diferença entre a propriedade class id é que+ a propriedade id suporta só um seletor.
