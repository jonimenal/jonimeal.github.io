Variantes


Variantes são basicamente uma caixinha que guarda algum tipo de dado.
A maneira de criares uma variante é simples!
Apenas precisas de usar o comando let e dar o nome que quiseres à tua variante um dos 7 tipos de dados dentro da dita variante.


  1. Boolean
  2. null
  3. undefied
  4. Number
  5. String
  6. Symbol
  7. Object
<script>

let nomeAtoa = 26

</script>

Funções


Funções são basicamente ações paradas.
Quando crias uma função, estás a criar uma ação que só vai acontecer por um trigger

O primeiro passo para criares uma função é escrevendo o comando function, de seguida, dás o nome que quiseres à tua função (no final do nome tens de colocar "()"). E por último escreves as "ações" que queiras que a tua função faça.

<script>

function nomeExemplo(){


document.getElementById("super").style.margin-left = "12px"

}
</script>

Depois de escreveres a tua função, tens de criar um trigger que acione a função. Um trigger é um tipo de propriedade que colocas dentro de um elemento HTML.
Para aplicares uma função em um elemento podes pensar como se estivesses a escrever uma frase:

Quando eu carregar no botão, a ação nomeExemplo() vai acionar.

<html>
<head>
</head>
<body>

<button onclick="nomeExemplo()"> Carrega em mim! </button>

</body>
</html>
