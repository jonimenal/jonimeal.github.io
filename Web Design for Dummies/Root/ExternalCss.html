External CSS


External CSS é a forma de escrever CSS em um documento à parte do HTML.

Para escrever External CSS é só preciso criar um ficheiro novo e escrever como se estivesses a escrever Internal CSS, a única diferença é que os setores não estão dentro de nenhum elemento.

Tal como o Internal CSS...
External CSS tem setores, regras e valores

.falafel{
margin: 20px 0px 10px 100px;
backgound-color: rgb(51, 51, 51);
}
p{
color: #ffffff;
backgound-color: 000000;
font-size: 24px;
font-family: Times New Roman;
}
#softIceCream{
padding: 5px;
backgound-color: #b09cfa;
width: 20px;
height: 2000px;
}

margin: 0px;
}
