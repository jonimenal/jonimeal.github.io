Inline CSS


Internal CSS é a forma de escrever CSS dentro de um elemento HTML.

Esta é a maneida de escrever CSS mais diferenciada das outras.
A forma que escreves Internal CSS é adicionando a propriedade style dentro do elemento HTML que pretendes estilizar.

Inline CSS tem a propriedade, regras e valores

<html>
<head>
</head>
<body>

<p style="font-family: Arial; font-size: Large; color: #f98b88;">Isto é um texto de exemplo :)</pt>

</body>
</html>
