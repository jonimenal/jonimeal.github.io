Seletor CLASSE


O seletor classe escolhe todos os elementos que estejam classificados com o seu nome.
A forma como eu escrevo um seletor tipo classe em CSS é colocando um "." antes do que eu escrevo.
Por exemplo: .palavra

<style>
.palavra{


}
</style>

Quando eu escrevo o meu seletor classe e coloco todas as regras que eu quero dentro dele... Vou ter de classificar algum elemento com o nome do seletor. Eu faço isso usando a propriedade class.

<html>
<head>
</head>
<body>

<p class="palavra">

</body>
</html>

Também podes ver pelo exemplo acima, quando escreves o valor da prorpiedade, não colocas ".", apenas o que escreveste depois.

Curiosidade: uma propriedade class consegue suportar mais do que um seletor.
